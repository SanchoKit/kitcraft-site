Container

Docker & Compose Setup

The entire server runs as a single container using the itzg/minecraft-server image with a bind-mounted data volume. 

services:
  mc-server:
    image: itzg/minecraft-server
    container_name: mc-server
    ports:
      - "25565:25565"   # Minecraft
      - "8100:8100"     # BlueMap
    environment:
      EULA: "TRUE"
      TYPE: "PAPER"
      VERSION: "1.21.1"
    volumes:
      - "./minecraft-data:/data"
    restart: unless-stopped
